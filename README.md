# Employee Analytics SQL Project

## Overview
This project analyzes employee, project, and data science team data for **ScienceQtech**, a data-driven company operating across multiple industries. The objective is to answer HR and business-related questions using SQL and demonstrate database design, querying, and optimization techniques.


## Dataset

The project uses three relational tables:

### 1. emp_record_table
Stores employee information including:
- Employee ID
- Name
- Gender
- Role
- Department
- Experience
- Country
- Salary
- Employee Rating
- Manager ID
- Project ID

### 2. proj_table
Contains project information:
- Project ID
- Project Name
- Domain
- Start Date
- Closure Date
- Development Quarter
- Project Status

### 3. data_science_team
Stores employee details for the Data Science department.
